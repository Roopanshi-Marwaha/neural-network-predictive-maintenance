# Neural Network-Based Predictive Maintenance

## Overview

This project develops an AI-powered predictive maintenance system that predicts machine failures using industrial sensor data. The system helps reduce downtime, maintenance costs, and unexpected equipment breakdowns in smart manufacturing environments.

## Features

* Data Preprocessing and Cleaning
* Exploratory Data Analysis (EDA)
* Random Forest Model
* Deep Neural Network
* XGBoost Model
* SHAP Explainability
* TensorFlow Lite Optimization
* Gradio Interface

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn
* TensorFlow
* XGBoost
* SHAP
* Gradio

## Dataset

AI4I 2020 Predictive Maintenance Dataset

## Results

The models were evaluated using standard classification metrics such as Accuracy, Precision, Recall, F1-Score, and ROC-AUC.

Among the implemented models, XGBoost achieved the best overall performance for machine failure prediction.
